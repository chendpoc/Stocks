# Paper Order Submit / Query / Cancel

Status: Blocked by Contract

## Blocker

Requires order state, position state, pre-trade checks, and broker-like mirror
even if not live trading.

## Required Contract Or Gate

Paper execution contract and risk policy.

## Boundary

Paper execution is still execution. It must not be treated as a harmless UI or
workflow extension.

## Unblock Step

Define paper account, order lifecycle, pre-trade risk checks, and cancellation
semantics.
