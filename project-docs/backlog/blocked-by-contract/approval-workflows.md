# Approval Workflows

Status: Blocked by Contract

## Blocker

Approval request schema, permission model, capability gate, and audit semantics
must exist first.

## Required Contract Or Gate

Shared Platform approval and capability policy.

## Boundary

Do not add approval buttons, implicit approvals, or workflow publication before
the policy contract exists.

## Unblock Step

Define approval request schema, approver identity, capability scope, and audit
output.
