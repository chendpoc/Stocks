# Agent Dev Archive

Historical `.agent-dev` execution artifacts live here so current agent entry
docs stay small.

Archived areas:

- `tasks/`: T001-T007 task files and slice plans.
- `specs/`: completed, superseded, or historical feature specs.
- `worker-prompts/`: old worker prompts for completed tasks.
- `reviews/`: historical review records and findings.
- `presentations/`: old development and review presentations.
- `design-notes/`: historical workflow and self-evolving-agent notes.

Do not use these files as default first reads. Read them only when a task
explicitly asks for historical implementation evidence.
