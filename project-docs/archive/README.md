# Archive

This directory keeps historical planning, migration, and UI prototype material.
It is not the current source of truth for implementation work.

Current trader-agent work is routed through:

- `project-docs/research-agent/target-system/trader-agent/README.md`
- `apps/trader-agent/backend`
- `apps/trader-agent/shared`
- `apps/trader-workflows`
- `apps/trader-cli`

Archived areas:

- `research-console/`: legacy research-console and research-workbench material.
- `trader-cockpit/`: removed Web cockpit PRD, design, plans, and implementation
  notes.
- `shared-agent-memory/`: completed M0-M6 implementation plans kept as history.
- `trader-agent-backend-reconciliation/`: superseded backend reconciliation
  worker prompts kept as history.
- `agent-dev/`: historical `.agent-dev` task/spec/review/presentation/worker
  artifacts, including T001-T007.
- `forward-intel/`: historical Forward Market Intelligence MVP plans. These
  mention old cockpit assumptions and are not current implementation guidance.
- `legacy-trader-agent/`: superseded trader-agent v0.3 system design with the
  old Web Cockpit layer.

Do not use archived documents as default first reads. Read them only when a task
explicitly asks for historical context or migration evidence.
