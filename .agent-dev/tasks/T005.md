## S1: Patch · B2/B3/D312 longbridgeCli 子命令兜底白名单

### 实施指南

1. `longbridgeCli.ts` 中 `validateLongbridgeInvoke`：在 alert/sharelist 现有 if 分支之后，加默认分支：

   ```ts
   const DEFAULT_ALLOWED_FIRST_ARGS = new Set([
     "list","show","get","detail","query","chain","history","snapshot",
     "search","summary","stats","info","rank","peers","calendar",
     "holders","actions","topics","rating","premium",
   ]);

   // 在已有 if (command === 'alert') / if (command === 'sharelist') 之后
   if (args.length > 0 && !args[0].startsWith("--")) {
     if (!DEFAULT_ALLOWED_FIRST_ARGS.has(args[0])) {
       return { ok: false, code: "FORBIDDEN_SUBCOMMAND",
                message: `子命令 ${args[0]} 不在 _default_allowed_first_args 兜底白名单` };
     }
   }
   ```

2. 确认 `BLOCKED_TOP_LEVEL` **不含** `check`（之前就不含，spec 已对齐）。
3. `longbridgeCli.test.ts` 加 3 个测试：
   - `expect(BLOCKED_TOP_LEVEL.has('check')).toBe(false)`
   - `validateLongbridgeInvoke({command:'option',args:['create-strategy']}).code === 'FORBIDDEN_SUBCOMMAND'`（被 blocked_arg_tokens 或 default 白名单兜住）
   - `validateLongbridgeInvoke({command:'check',args:[]}).code === 'NOT_WHITELISTED'`（check 不在 GATEWAY_WHITELIST）

const QUOTE_PARAMS = z.object({

## S5: Test · M3 新建 longbridgeTools.test.ts

### 实施指南

```ts
import { describe, it, expect } from "vitest";
import { createLongbridgeTools } from "./longbridgeTools.js";
import specJson from "../../../../.agent-dev/specs/trader-longbridge-agent-cli/spec.json";

describe("longbridgeTools", () => {
  it("共 23 个工具（22 Tier1 + 1 invoke）", () => {
    const tools = createLongbridgeTools();
    expect(Object.keys(tools)).toHaveLength(23);
  });

  it("Tier1 工具名与 spec.json 一致", () => {
    const tools = createLongbridgeTools();
    const names = Object.keys(tools).filter(n => n !== "longbridgeInvoke");
    const expected = specJson.tier1_tools.map((t: any) => t.tool_name).sort();
    expect(names.sort()).toEqual(expected);
  });

  it("longbridgeInvoke 拒绝 order", async () => {
    const tools = createLongbridgeTools();
    const result = await tools.longbridgeInvoke.execute({ command: "order", args: [] }, {} as any);
    expect(result.ok).toBe(false);
    expect(result.code).toBe("FORBIDDEN_COMMAND");
  });

  it("getLongbridgeQuote symbols>10 返回 MULTI_SYMBOL_LIMIT", async () => {
    const tools = createLongbridgeTools();
    const symbols = Array.from({ length: 11 }, (_, i) => `T${i}`);
    const result = await tools.getLongbridgeQuote.execute({ symbols }, {} as any);
    expect(result.ok).toBe(false);
    expect(result.code).toBe("MULTI_SYMBOL_LIMIT");
  });

  it("longbridgeInvoke check 返回 NOT_WHITELISTED", async () => {
    const tools = createLongbridgeTools();
    const result = await tools.longbridgeInvoke.execute({ command: "check", args: [] }, {} as any);
    expect(result.ok).toBe(false);
    expect(result.code).toBe("NOT_WHITELISTED");
  });
});
```
